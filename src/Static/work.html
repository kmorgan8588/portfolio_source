Projects

  • Zilhome

    Proxy Microservice Remastered

    An open source home listing service. The proxy page supports five mircorservices which populate the page with all necessary information. All mircorservices were built from the ground up using React.js, HTML, and CSS. The Servers are powered by Node.js and Express, with each micrcoservice being handled by a different server and database. Check out the repos to see what I had a hand in. The remastered version takes the Node/Express server and reinvisions it in C# and the .NET framework.

    Deployed with Docker to Amazon Web Services using EC2 instances.
    Status: Currently not deployed

  • Better Homes and Hackers

    Proxy Microservice

    An inherited code base project. The primary effort was to revamp the backend infrastructure with an emphasis on building out a scalable system during deployement. All microservices were scaled individually and by hand as a learning exercise in system design.

    Deployed with Docker to Amazon Web Services using EC2 instances.
    Structure:

    • Load Balancer - Nginx
    • Database - Postgres on one instance
    • Servers - Three Node/Express
    • Caching - not implemented before project time limit
    • Testing - Loader.io and NewRelic

    Status: Currently not deployed

  • Dice Dice Baby

    Project is currently private, checkout my Github for more.

    A dice rolling application built from the ground up using React.js with Material UI. Allows the user to select from various types of dice to roll. With a persistent rolling history, powered by MongoDB/Mongoose, with the option to turn back time and re-roll for a *hopefully* better result from directly in the history.
    Status: Currently not deployed
